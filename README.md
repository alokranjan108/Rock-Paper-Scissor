# Rock-Paper-Scissor
A beginner-friendly Rock Paper Scissors game built in Python with a clean multi-file structure. Play against a bot in your terminal.
## 📁 Project Structure
 
```
├── main.py         # Entry point — runs the game loop
├── game.py         # Core logic — decides who wins each round
├── bot.py          # Bot AI — randomly picks a move
├── display.py      # All UI / print statements
├── scoreboard.py   # Tracks and updates scores
└── constants.py    # Shared values (choices, emojis, win rules)
```
## 🧠 Concepts Used
- Functions and return values
- Dictionaries and conditionals
- `while` loop for the game loop
- `import` across multiple files (modules)
- `random.choice()` for bot logic
- Input validation
